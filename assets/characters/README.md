# Character portraits

Drop character art here using the `id` from the matching `.tres` as the
filename. Current IDs:

- `capy_brown.png`
- `capy_chef.png`
- `capy_zoomer.png`

## Format

Same as fruits, just larger:

| Property | Recommended |
|---|---|
| Format | **PNG** with alpha (also `.webp`, `.jpg`, `.svg`) |
| Background | Transparent |
| Dimensions | Square, **256×256** or **512×512** |
| Aspect | Doesn't have to be exactly square — the sprite is auto-fit so its longest side ≈ 120 px in-match, and ≈ 200 px on the select card |

If a character has no image file the colored square placeholder is shown
instead, so you can add art one character at a time.
